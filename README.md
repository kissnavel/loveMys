# loveMys(Miao-Yunzai & TRSS-Yunzai)

## 安装插件
在云崽根目录下打开终端，运行：
``` bash
git clone https://github.com/kissnavel/loveMys.git ./plugins/loveMys-plugin/
pnpm install --filter=loveMys-plugin
```

## 使用插件
重启后在 config/api.yaml中填入api、resapi、key、query
